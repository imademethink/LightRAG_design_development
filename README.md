# Steps
As explained in above YouTube video, install Ollama (Windows/ Linux) -> Sign In

Download cloud AI model using: 
ollama run gpt-oss:120b-cloud (or your preferred AI model)

git clone https://github.com/imademethink/LightRAG_design_development.git

cd LightRAG_design_development-master

pip install -r requirements.txt

python light_rag_0.py
